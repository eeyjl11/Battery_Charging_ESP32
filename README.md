# Battery_Charging_ESP32
 ESP32 S3 with integrated li-ion battery charging.
 
 ## MCP73831T-5ACI_OT
 For charging battery.
 
 ## DW01A and FS8205
 Handles battery protection.
 
 ## AMS1117-3.3
 3.3V Voltage regulator for ESP32.
